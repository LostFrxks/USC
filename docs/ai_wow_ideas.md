# AI Wow Ideas (Shortlist)

Дата фиксации: 2026-03-03

## 1) What-if Симулятор
- ИИ-симулятор с интерактивными слайдерами по ключевым параметрам:
  - цена SKU,
  - доля поставщика,
  - promo intensity,
  - SLA доставки.
- Результат пересчитывается сразу на экране:
  - выручка,
  - маржа,
  - cancel rate,
  - delivery rate,
  - риск концентрации.
- Цель: дать эффект "живого бизнес-двойника" в реальном времени.

## 2) Ask -> Auto Chart + Drilldown
- Пользователь задает вопрос на естественном языке:
  - "покажи, почему просела выручка в марте".
- ИИ сам строит нужный график и автоматически открывает разрезы:
  - категория -> SKU -> поставщик -> заказы.
- Цель: убрать ручной поиск в аналитике и сделать объяснимую визуальную аналитику "по запросу".
